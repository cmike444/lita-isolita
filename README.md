# lita-isolita

Make Lita do more work

## Installation

Add lita-isolita to your Lita instance's Gemfile:

``` ruby
gem "lita-isolita"
```

## Usage

    *# Mind Manners*
      Chris: Lita, do some fucking work
      Lita: Mind your manners Chris.. 

    *# Echo*
      Chris: Lita echo Bacon ipsum dolor amet
      Lita: Bacon ipsum dolor amet
